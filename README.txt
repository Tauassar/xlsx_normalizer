Скрипт для приведения отчета СУО в вид, пригодный для загрузки в Axon и дальнейшей сверки
В папке dist/ содержится .exe билд, для запуска скрипта с исполняемого файла

При необходимости изменения настроек, необходимо открыть файл settings.py:
DEBUG = Устанавливает уровень логирования
max_search_index = максимальная строка/столбец для поиска значений из search_strings
first_row = с какой строки начинается заполнение данными
first_col = с какого столбца начинается заполнение данными
fact_sheet_name = Наименование листа
search_strings = Наименование столбцов по которым ведется поиск
